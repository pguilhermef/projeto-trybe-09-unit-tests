# projeto-trybe-09-unit-tests

Eu implementei várias funções na resolução dos requisitos propostos e/ou testes unitários para garantir que as implementações das funções estão corretas, de acordo com o que está sendo solicitado em cada enunciado.

Nesse projeto, foi necessário conhecimento para:

Escrever testes unitários utilizando o módulo Jest do NodeJS para verificar o correto funcionamento das funções;
Escrever funções de forma que elas atendam a testes já implementados;
Escrever testes e funções utilizando uma abordagem de desenvolvimento orientado a testes.
